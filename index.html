What this Hackathon is about:

Build web-apps from scratch

We will be focussing on the basic building blocks of the web.

From scratch means we will start off without using any fancy libraries to facilitate a broader understanding of how HTML, CSS ans JS actually work and combine to form the basis of everyhting you see on the web.

During most of the hackathon, we will focus on making this page look and feel more modern.

A broad outline of the technologies we will be using:

  • HTML and content,
  • CSS and responsive design,
  • Javascript to make your web-page dynamic.
What you will need

We will be using VSCode as our text editor / IDE, but feel free to use whatever tools you prefer. Instructions and specific help with tools will have to be limited to VSCode though.

We will be using Google Chrome as our desktop browser, to ensure the features we need available on the browser. Other browsers may or may not have similar features. Chrome on mobile may also be usefull depending on the features you want to build into your web-app.

As all webpages need to be hosted somewhere, we will also look at how GitHub pages allows serving of static websites without the need to set up a server.

Depending on time and interest we may also look into some server-side Javascript (using NodeJS) and some native Javascript (using react-native and expo.io).
